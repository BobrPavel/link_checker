# Бот для проверки URL ссылок
Это Telegram бот для проверки безопасности URL ссылок. Создавал его с целью демонстрации собсвенных возможностей,
и поэтому ревлизовал большое количество разных функций.

## Функционал
1) Проверка URL через базы фишинговых и вредоносных сайтов, а так же в чёрных списках.
2) Проверка SSL / HTTPS и инфраструктуры
3) Вывод истории домена, даты регистрации и регистратора
4) Аналитика ссылок на маскировку, турбоскватинг, подсчёт количества внешних и внутренних ссылок, а так же поиск трекинговых парметров.
5) Система рейтинга
6) Кеширование проверенных URL, для обеспечения более эффективного использования сторонних API

## Локальное развёртывание проекта
Клонируйте репозиторий проекта и перейдите в него с помощью командной строки
```
git clon https://github.com/BobrPavel/link_checker.git

```

Создайте виртуальное окружение Pyton и активируйте его
```
python3 -m venv venv  
```
```
venv\Scripts\activate  
```
Установите зависимости

```
pip install -r requirements.txt  
```
Создайте файл .env и запишите туда ваш Telegram токен и AI API key/
```
TOKEN=Ваш токен
OPENAI_KEY=Ваш GPT API KEY
GOOGLE_SAFE_BROWSING_KEY=Ваш Google safe beowsing key
VIRUSTOTAL_KEY=Ваш VirusTotal key
```

## Запуск бота:
Запустите файл bot.py и любое приватное сообщение боту.
